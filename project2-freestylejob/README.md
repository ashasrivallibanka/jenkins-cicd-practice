# Project 2 – Jenkins Freestyle Job

## Objective

Learn how to create and execute a Jenkins Freestyle Job.

## What is a Jenkins Job?

A Jenkins Job is a set of instructions executed by Jenkins to automate tasks such as building code, testing applications, or deploying software.

## What is a Freestyle Job?

A Freestyle Job is the simplest type of Jenkins job. It allows users to execute shell commands or build steps through a graphical interface without writing a pipeline script.

## Commands Used

```bash
echo "Hello Jenkins"
whoami
pwd
date
uname -a
ls -la
```
