# Project 1 - Jenkins Installation using Docker

## Objective

Install Jenkins using Docker and understand its basic architecture.

## Tools Used

- Ubuntu (WSL)
- Docker
- Jenkins LTS

## Steps Performed

1. Created Jenkins practice repository.
2. Started Jenkins using Docker.
3. Verified Docker container.
4. Accessed Jenkins Web UI.
5. Retrieved initial admin password.
6. Installed suggested plugins.
7. Created administrator account.
8. Explored Jenkins dashboard.

## Output

- Jenkins running successfully.
- Web dashboard accessible.
- Plugins installed.
- Admin account created.
